# mbta

## Task: Replicate North Station Train Announcement Board

### Installation

```
> git clone git@github.com:raisch/mbta.git
> cd mbta
> npm install
```

### Run
```
> npm start
```

### Build Documentation

```
> npm run docs
```

### View Documentation

#### OSX

```
> open ./out/index.html
```

#### Other OSen

Open browser of choice on `./out/index.html`
